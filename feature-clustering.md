# AI-FOS — Feature Clustering (Must / Should / Could)

> "Everything feels like a Must. It's not. Most things are a Could in disguise."

---

## 🔴 MUST
*Core user flow — without these, there's no product.*

- Unified Dashboard
- Spending Agent (expense tracking)
- AI Orchestrator (plain-language summary)

**Target:** 3 Musts. If you have 8, you have 0.

---

## 🔵 SHOULD
*Adds value, but not critical at launch.*

- Investment Agent
- Tax Agent

**Target:** Week 1 post-workshop priority.

## ⚫ COULD
*Nice to have. Ideas, not features — not yet.*

- Loan Agent
- Fraud Detection Agent
- Goal Planning Agent
- Economic Intelligence Agent
- Voice Assistant / OCR Scanner
- Mobile App

**Target:** Maybe never. Revisit later.
